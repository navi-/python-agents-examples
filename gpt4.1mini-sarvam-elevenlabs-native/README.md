# GPT-4.1 Mini + Sarvam + ElevenLabs Voice Agent (Native)

Voice agent using OpenAI GPT-4.1 mini for LLM, Sarvam.ai for speech-to-text, ElevenLabs for text-to-speech, with Plivo telephony and Silero VAD for turn detection. Native orchestration (no framework).

Full documentation will be generated with `/document-example`.
